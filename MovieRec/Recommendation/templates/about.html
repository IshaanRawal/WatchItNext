Welcome to Watch It Next!


Watch it Next is a Movie Recommendation website that is built by building the logic in Python and supported by Flask, HTML and CSS Bootstrap. It applies the principles of Maachine Learning to raw Movie Review files and converts it into valuable information!


Importance of Algorithms


Any Recommendation Engine thrives on certain indispensible algorithms for it to work in an efficient manner. It would include various search algorithms to query its large database and return a specific asked value. It also includes sorting algorithms to efficiently return certain data in usable form. These algortihms make up the engine and we'll be talking about some in detail in the next module.


Algorithms used in Watch It Next


Watch It Next uses certain in-built functions to perform Search and Sort. Tim Sort is used while sorting the list of Recommendations based on the nearest distance to the searched movie. Tim Sort is a special derivation of Merge Sort and Insertion Sort. It works well while dealing with Real-World data. In the worst case scenario, Tim sort takes O(logn) time to sort. It also takes lesser memory allocation and comparisions as compared to Quick Sort


KNN Algorithm


The K Nearest Algorithm is the most vital algorithm for the functioning of Watch It Next. It is one of the simplest Supervised Machine Learning Algorithm. It is mostly used for classification. KNN ALgorithm in Watch It Next calculates the nearest related movies corresponding to the Searched Movie using the parameter of Cosine Distance. The model is trained using the filtered data converted into a Compressed Sparse Row Matrix or CSR_Matrix as defined in the code. This helps to reduce the noise from the data and efficiently train the KNN Algorithm. The model returns 10 Nearest movies in reference to the Searched movie.
